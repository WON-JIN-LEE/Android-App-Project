# 음주측정 App

### Project

- 개발인원 : 3명
- 개발 툴 : Android Studio
- 언어 : JAVA, XML
- 개발 기간: 2019.06.01. ~ 2019.06.24

- 음주측정 디바이스와 블루투스를 연결할 수 있습니다.
